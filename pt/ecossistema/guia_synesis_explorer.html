1 Guia rapido: Synesis Explorer (VS Code)

Objetivo: ajudar voce a instalar e usar a extensao para navegar seus arquivos Synesis (.syn, .synt, .synp, .syno) de forma simples.

1.1 O que a extensao faz

  • Mostra suas referencias, codigos e relacoes em paineis laterais
  • Abre um grafo de relacoes e o resumo (abstract) das referencias
  • Ajuda a encontrar trechos com facilidade

1.2 Requisitos basicos

  • VS Code instalado
  • Arquivos Synesis no seu computador

1.3 Instalacao (arquivo VSIX)

  1. Abra o VS Code
  2. Clique no icone de extensoes (barra lateral)
  3. Clique nos tres pontinhos (menu) e escolha ‚ÄúInstall from VSIX‚Ä¶‚Äù
  4. Selecione o arquivo .vsix da extensao
  5. Aguarde a instalacao e reinicie o VS Code se solicitado

1.4 Como abrir seu projeto

  1. Abra a pasta do seu projeto no VS Code
  2. Certifique-se de que seus arquivos .syn e o projeto .synp estao na pasta

1.5 Onde encontrar o Synesis Explorer

  1. Na barra lateral esquerda, clique no icone ‚ÄúSynesis Explorer‚Äù
  2. Voce vera tres secoes:
    • References
    • Codes
    • Relations (aparece quando houver relacoes)

1.6 Usando os Explorers

1.6.1 References

  • Lista suas fontes (SOURCE)
  • Use o botao de atualizar (refresh) quando mudar arquivos
  • Use o filtro para encontrar rapidamente uma referencia

1.6.2 Codes

  • Mostra os codigos encontrados nos seus dados
  • Use refresh e filtro da mesma forma

1.6.3 Relations

  • Mostra relacoes (chains) encontradas
